About

Welcome to the Power BI Dashboard Viewer. This platform provides easy access to interactive Power BI dashboards with a clean, responsive interface.

Features

  • Browse dashboards in an organized, card-based layout
  • View interactive Power BI reports embedded in the browser
  • Download datasets and Power BI project files (.pbix)
  • Mobile-responsive design for viewing on any device
  • Simple admin interface to add new dashboards

How to Use

  1. Browse the available dashboards on the home page
  2. Click "Open Dashboard" to view a specific dashboard
  3. Interact with the embedded Power BI report
  4. Download datasets or project files if available

Adding New Dashboards

Administrators can add new dashboards using the "Add Dashboard" link in the navigation menu. You'll need to provide the Power BI embed URL, optional dataset and project file URLs, and metadata like name, description, and tags.
